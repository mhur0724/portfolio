Natural Events Tracker

Live Site GitHub

Built With: React, CSS, API

A React based Natural Events Tracker (inspired by TraversyMedia), but I've added the ability to filter the events that you'd like to see.

It utilizes both the Google Maps API and NASA's Event Tracker API in order to return a map with markers and information of the filtered event(s) that have taken place over the globe.

Built With: jQuery, JavaScript, CSS, HTML, API

This is a pokedex I made that uses fetch to call a Pokemon API. This project is meant to be jQuery/JavaScript centered so it is not responsive.

To use it, you can enter either the name or the ID of the pokemon. The API can have some unexpected names with added information like "Pumpkaboo-average" so you may want to defer to using pokemon's ID if you cannot find your desired Pokemon.

Studio Ghibli 3D Still

Howl's Moving Castle
Kiki's Delivery Service
Princess Mononoke
Whisper of The Heart
The Wind Rises

This project itself isn't a web development project, but this card implements a bootstrap carousel to display these 3D stills.

I made these for my friends who all love Ghibli movies. A lot of hours went into this but I think it turned out great!

With the exception of the more complex STL files (lamp post, log, house, etc). I designed everything on CAD, 3D printed the files on Form 3 printers, and painted everything using a mixture of acrylic and spray paint.
