Pokedex

Live Site Github

This is a pokedex I made using fetch to call a Pokemon API. I wanted to focus on using JS so it is not responsive. To use it, you can enter either the name or the ID of the pokemon. The API can have some unexpected added information like "Pumpkaboo-average" so you may want to defer to using ID's if you cannot find your desired Pokemon.

It currently has a bug with the evolution images section of the pokedex. For whatever reason it occasionally mixes up the order of the evolution chain. I believe I need to add async/await somewhere in the code.

This project itself isn't a web development project, but this card implements a bootstrap carousel to display these 3D stills.

I made these for my friends who all love Ghibli movies. A lot of hours went into this but I think it turned out great!

I used CAD to design everything, 3D printed everything on Form 3 SLA printers, and painted everything with acrylic and spray paint.
